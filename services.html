Manuscript Editing

Deep Edit Re-read Light Edit Complete Edit Proofread Sample Edit Web Design

Deep Edit

Two pass report plus relevant in-manuscript mark-up focusing on story, structure, style; including characters, POV, dialogue, word choice, pacing, lose threads/inconsistencies etc. This takes the form of first pass–author review–second pass. - £0.02/word

Re-read

Akin to a third/fourth/etc. pass, but read-based only, i.e. up to four pages of notes are provided but no actual edits performed. Wouldn’t it be nice to be able to know if all those changes you made helped, and be able to check for that as many times as you like? The re-read can do it for you. - £0.003/word

Light Edit

In-manuscript mark-up focussing on grammar, spelling, punctuation, fact checking, formatting. I recommend a Light Edit only after deep level editing has been completed. - £0.014/word

Complete Edit

– Combined Deep and Light Edit, delivered in stages: The Light Edit follows a completed Deep Edit. When you book these together as the Complete Edit I offer you a discounted price. - £0.028/word

Proofread

– Final pass check for typos and other strays. - £0.007/word

Sample Edit

First 10 pages at 250 words/page. A sample edit allows us both to see whether we are a good fit and what level of edit is required. - £25

Web Design

If you like the idea of having a website to accompany and support your creation, consult with me and I will code one from scratch for you.


Other Editing

Copy Edit Line Edit Combined Copy and Line Edit Proofread

Copy Edit

Focuses on structural suggestions; think of it as the bigger picture or macro edit. - £0.015/word

Line Edit

Focuses on grammar, spelling, punctuation; no content/structural edits. Think of it as the micro edit. - £0.015/word

Combined Copy and Line Edit

When you book the Copy and Line edit together I offer you a discounted price. - £0.02/word

Proofread

This should be the very final stage after the work has been put through all relevant edits. If the required changes are many, the service will be classed as a line edit instead of proofread. - £0.007/word


A minimum charge of £10 applies to all ‘Other Editing’ services.
